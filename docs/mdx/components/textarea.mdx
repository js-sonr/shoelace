## Examples

### Labels

Use the `label` attribute to give the textarea an accessible label. For labels that contain HTML, use the `label` slot instead.

<Tabs items={['Preview', 'Web Component', 'React']}>
  <Tab value="Preview">
    ```tsx
    <nu-textarea label="Comments"></nu-textarea>
    ```
  </Tab>
  <Tab value="Web Component">
    ```html
    <nu-textarea label="Comments"></nu-textarea>
    ```
  </Tab>
  <Tab value="React">
    ```tsx
    import SlTextarea from '@onsonr/nebula/dist/react/textarea'
    ;
    
    const App = () => <NuTextarea label="Comments" />;
    ```
  </Tab>
</Tabs>

### Help Text

Add descriptive help text to a textarea with the `help-text` attribute. For help texts that contain HTML, use the `help-text` slot instead.

<Tabs items={['Preview', 'Web Component', 'React']}>
  <Tab value="Preview">
    ```tsx

    ```
  </Tab>

### Prevent Resizing

By default, textareas can be resized vertically by the user. To prevent resizing, set the `resize` attribute to `none`.

<Tabs items={['Preview', 'Web Component', 'React']}>
  <Tab value="Preview">
    ```tsx
    <nu-textarea resize="none"></nu-textarea>
    ```
  </Tab>
  <Tab value="Web Component">
    ```html
    <nu-textarea resize="none"></nu-textarea>
    ```
  </Tab>
  <Tab value="React">
    ```tsx
    import SlTextarea from '@onsonr/nebula/dist/react/textarea'
    ;
    
    const App = () => <NuTextarea resize="none" />;
    ```
  </Tab>
</Tabs>

### Expand with Content

Textareas will automatically resize to expand to fit their content when `resize` is set to `auto`.

<Tabs items={['Preview', 'Web Component', 'React']}>
  <Tab value="Preview">
